# PSPI---Parity-Serial-Peripheral-Interface
- A new and improved version based on the SPI protocol
- Added Parity Bit from Master to Slave
- Added Acknowledgement Bit from Slave to Master
- Error Checking
- Retransmission
- Tested for upto 12.5MHz on the Boolean Board (Spartan7)

Note - The design might not work as expected in the simulation.
